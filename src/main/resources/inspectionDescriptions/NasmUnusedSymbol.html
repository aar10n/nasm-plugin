Reports symbols that are defined but never used.

Detects unused:

  • Labels (both global and local)
  • Macros (%define, %macro)
  • Constants (%assign, equ)
  • Data labels

Example:


    unused_label:          ; Warning: never used
        nop

    %macro UNUSED 1        ; Warning: never invoked
        mov rax, %1
    %endmacro

    UNUSED_CONST equ 42    ; Warning: never referenced

Note: Entry points like _start and main are excluded. Symbols declared with global are also excluded as they may be used externally.
