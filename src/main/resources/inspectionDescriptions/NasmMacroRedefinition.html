Reports when a macro is defined multiple times.

In NASM, redefining a macro silently replaces the previous definition, which can cause unexpected behavior.

Example:


    %define FOO 1           ; Original definition
    %define FOO 2           ; Warning: already defined at line 1

    %macro test 0
        nop
    %endmacro
    %macro test 0           ; Warning: already defined at line 3
        ret
    %endmacro

Multi-line macros with different parameter counts are considered distinct: 


    %macro print 1          ; OK
        mov rax, %1
    %endmacro
    %macro print 2          ; OK - different parameter count
        mov rax, %1
        mov rbx, %2
    %endmacro

Note: Does not apply to %assign, which is designed for reassignment.
